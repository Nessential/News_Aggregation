### 任务
基于提供的新闻证据，回答用户问题。

### 证据

### 输出要求
1. 只输出严格 JSON，不要输出任何解释。
2. 必须输出 `answerItems` 数组。
3. 每个 item 必须包含：
   - `text`: 单条回答文本，**必须以（1）、（2）、（3）...形式编号分点**
   - `newsIds`: 该条回答关联的新闻 ID 字符串数组（如 ["76","102"]）
4. **在回答中必须包含对应新闻的发布时间**，格式如："（1）[2024-01-15] ..."
5. 若证据为空或无法回答，也要输出 `answerItems`，但 `newsIds` 可为空数组。
6. 禁止输出旧字段：`answer`、`citations`。

### 输出格式
{
  "answerItems": [
    {
      "text": "（1）[发布时间] 回答内容...",
      "newsIds": ["76", "102"]
    }
  ]
}
