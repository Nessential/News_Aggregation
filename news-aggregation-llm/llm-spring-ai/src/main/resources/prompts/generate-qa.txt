### 输出要求
1. 只输出严格 JSON，不要输出任何解释。
2. 必须输出 `answerItems` 数组。
3. 每个 item 必须包含：
   - `text`: 单条回答文本
   - `newsIds`: 该条回答关联的新闻 ID 字符串数组（如 ["76","102"]）
4. 若证据为空或无法回答，也要输出 `answerItems`，但 `newsIds` 可为空数组。
5. 禁止输出旧字段：`answer`、`citations`。

### 输出格式
{
  "answerItems": [
    {
      "text": "...",
      "newsIds": ["76", "102"]
    }
  ]
}
