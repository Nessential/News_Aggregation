### 输出要求
1. 只输出严格 JSON，不要输出任何解释。
2. 必须输出 `answerItems` 数组。
3. 每个 item 必须包含：
   - `text`: 一条摘要结论，**必须以（1）、（2）、（3）...形式编号分点**，并**包含新闻发布时间**
   - `newsIds`: 支撑该结论的新闻 ID 字符串数组
4. 禁止输出旧字段：`answer`、`citations`。

### 输出格式
{
  "answerItems": [
    {
      "text": "（1）[2024-01-15] 摘要内容...",
      "newsIds": ["76"]
    }
  ]
}
